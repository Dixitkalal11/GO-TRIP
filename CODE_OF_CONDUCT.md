# Contributor Covenant Code of Conduct

This project adheres to the Contributor Covenant. By participating, you agree to the terms described here.

Standards
- Be respectful and constructive.
- No harassment, hate speech, or abusive behaviour.

Reporting
- Report issues to the maintainers via GitHub issues if you observe unacceptable behaviour.

For the full Code of Conduct, please refer to: https://www.contributor-covenant.org/version/2/0/code_of_conduct/
